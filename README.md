# Rails AI Workspace

An opencode workspace for Ruby on Rails development with orchestrated planning for large features and direct execution for small tasks.

## Structure

```
docs/           # Living design documents
  PRD.md        # Product requirements document
  architecture.md # Orchestration layer design

specs/          # Specifications
  skills.md     # Skills system spec
  plugins.md    # Plugins system spec

skills/         # Skill packages (SKILL.md + examples + scripts)
plugins/        # Plugin packages (PLUGIN.md + scripts)
```
